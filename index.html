jeDWM - jedzoka's Window Manager

Installation 1. Download the zip file and extract it to your home directory(~)
2. Open the terminal and type:
sudo apt-get install tinywm && sudo mv ~/jeDWM /usr/bin/tinywm
3. Logout, click the gear icon and select tinywm

Keybindings alt+shift+q logout
alt+z opens terminal
alt+b opens web browser
alt+p opens file manager
alt+x closes window
alt+right mouse button - resizing window
alt+left mouse button - moving window

changing layouts
alt+t tiling
alt+f floating
alt+m monocle

playing with tags - virtual desktops
alt+tab goes to the previous tag
alt+[1-9] goes to given tag
alt+shift+[1-9] moves window to given tag
alt+0 shows all opened windows
alt+shift+0 moves window to every tag
alt+shift+ctrl+[1-9] hides window on given tag
alt+ctrl+[1-9] shows content of given tag in current tag

Positioning windows
alt+enter
alt+h
alt+l
alt+i
alt+d

Changing focus
alt+j
alt+k

To add custom keybindings you need to recompile or make aliases in ~/.bashrc. Additionally you can use a tool like DConf
To check battery life and date add date && cat /sys/class/power_supply/BAT0/capacity to your ~/.bashrc

Compilation I just downloaded the source code and removed most instances of "bar" and "title" from dwm.c
That's why mine is smaller than default dwm build
Type sudo apt-get install make libxinerama-dev libx11-dev tcc
Then download dwm, cd into dwm and type:
sudo make -f Makefile && sudo mv dwm /usr/bin/tinywm
in config.mk you may need to set CC=tcc
